# 🛡️ VPN & Proxy Setup Guides

Практические инструкции по настройке инструментов обхода блокировок на собственном VPS.
Все гайды основаны на реальном опыте и актуальны для 2025–2026 года.

> Инструкции ориентированы на пользователей в России, где с февраля 2026 года
> Telegram замедлен, а VPN-трафик активно фильтруется через ТСПУ.

---

## 📋 Содержание

| Инструкция | Протокол | Транспорт | Для чего |
|---|---|---|---|
| [MTProxy + Fake TLS](./mtproxy-fake-tls.md) | MTProto | TCP | Telegram без VPN |
| [Hiddify Manager](./hiddify-setup.md) | VLESS/Reality, Hysteria2, TUIC | TCP/UDP | Полноценный VPN |
| [AmneziaWG 2.0](./amneziawg-setup.md) | WireGuard (модифицированный) | UDP | Резервный VPN |

---

## 🔍 Какой инструмент выбрать

### MTProxy + Fake TLS
Если нужен **только Telegram** — без установки VPN на устройство.
Прокси настраивается прямо в Telegram, работает автоматически при каждом запуске.

- ✅ Не нужно включать VPN вручную
- ✅ Работает на всех устройствах где есть Telegram
- ✅ Минимальные ресурсы на сервере
- ❌ Только для Telegram (не для браузера и других приложений)
- ❌ Звонки не работают через MTProxy

### Hiddify Manager
Если нужен **полноценный VPN** для всего трафика — сайты, приложения, всё.

- ✅ 20+ протоколов включая VLESS+Reality и Hysteria2
- ✅ Удобная панель управления
- ✅ Можно создавать пользователей для семьи/друзей
- ✅ Поддержка подписок (один QR-код — все конфиги)
- ❌ Занимает порты 80 и 443, нужен отдельный сервер

### AmneziaWG 2.0
Если нужен **резервный VPN** на случай блокировки основного.
Использует UDP вместо TCP — когда режут Xray/Reality, WireGuard часто продолжает работать.

- ✅ Другой транспорт (UDP), не пересекается с Hiddify
- ✅ Устанавливается через приложение без ручных команд
- ✅ Маскировка под QUIC/DNS
- ❌ Требует отдельного приложения на устройстве

## 🏗️ Рекомендуемая схема

Оптимальная связка для максимальной отказоустойчивости:

```
Сервер 1 (Россия, любой хостер)
└── MTProxy Fake TLS        → Telegram без VPN, порт 443
└── Uptime Kuma             → мониторинг, порт 3001

Сервер 2 (любая локация)
└── Hiddify Manager         → основной VPN (Reality, Hysteria2)
└── AmneziaWG 2.0           → резервный VPN (UDP)
```

**Почему MTProxy на российском сервере:** ТСПУ фильтрует трафик на границе сети.
Если прокси стоит в России, трафик до него не фильтруется — и уже замаскированным
уходит к серверам Telegram за рубеж.

## ⚙️ Требования

Для всех инструкций нужен VPS:

| Параметр | Минимум |
|---|---|
| ОС | Ubuntu 22.04 / 24.04 |
| Виртуализация | KVM |
| CPU | 1 ядро |
| RAM | 1 ГБ (2 ГБ для Hiddify) |
| Диск | 10 ГБ |
| Доступ | SSH (root) |
