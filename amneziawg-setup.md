# AmneziaWG 2.0 на своём VPS (Self-hosted)

## Зачем это нужно

AmneziaWG 2.0 — модифицированный WireGuard с маскировкой трафика. В отличие от
обычного WireGuard, который легко детектируется по UDP-сигнатуре, AmneziaWG
маскирует трафик под QUIC, DNS и другие UDP-протоколы.

Если основной VPN (например, Hidiffy/Xray) заблокируют — AmneziaWG продолжит
работать, потому что использует другой транспорт:

| | Hidiffy (Xray) | AmneziaWG 2.0 |
|---|---|---|
| Транспорт | TCP | UDP |
| Маскировка | HTTPS / TLS | QUIC / DNS |
| Блокируется | По TLS-сигнатуре | По UDP / QUIC |

Именно поэтому иметь оба — правильная стратегия резервирования.

Официальная документация: https://docs.amnezia.org/ru/documentation/instructions/new-amneziawg-selfhosted

---

## Требования к VPS

- ОС: Ubuntu 22.04 / 24.04 или Debian 12/13
- Виртуализация: KVM (OpenVZ/LXC не подойдут)
- Архитектура: x86-64 (arm64 не поддерживается)
- RAM: 1 ГБ и выше
- Диск: 10 ГБ и выше
- Публичный IPv4-адрес
- SSH-доступ (root или sudo)
- Ядро Linux: 4.14 и выше (для AmneziaWG 2.0)

Проверить версию ядра на сервере:

```bash
uname -r
```

## Шаг 1. Скачай приложение AmneziaVPN

Вся установка делается через приложение на твоём устройстве — на сервер
заходить руками не нужно. Amnezia сама подключится по SSH и всё настроит.

Скачай с официального сайта: https://amnezia.org/ru/downloads

Доступны версии для Windows, macOS, Android, iOS.

> Если сайт не открывается — используй зеркало:
> https://storage.googleapis.com/amnezia/amnezia.org?m-path=/ru/downloads

Минимальная версия для поддержки AmneziaWG 2.0: **4.8.12.9**

---

## Шаг 2. Добавь сервер в приложение

1. Открой AmneziaVPN
2. Нажми **➕** (плюс) или кнопку **Приступим**
3. Выбери **Self-hosted VPN**
4. Введи данные сервера:
   - IP-адрес: IP твоего VPS
   - Порт: `22` (стандартный SSH, менять не нужно)
   - Имя пользователя: `root`
   - Пароль: пароль от сервера
5. Нажми **Подключиться**

Приложение подключится к серверу по SSH и автоматически установит Docker
и все необходимые компоненты. Это займёт несколько минут.

---

## Шаг 3. Установи протокол AmneziaWG 2.0

После подключения к серверу:

1. Нажми на название подключения
2. Нажми на иконку ⚙️ (шестерёнка) справа от подключения
3. В списке протоколов найди **AmneziaWG**
4. Нажми на иконку 📥 справа от него
5. Нажми **Установить**

Установка займёт несколько минут. После завершения протокол появится
в списке доступных для подключения.

> Если на сервере уже был установлен AmneziaWG 1.x — он отобразится
> как **AmneziaWG Legacy**. Это нормально, оба работают независимо.

---

## Шаг 4. Подключись и проверь

1. В приложении выбери протокол **AmneziaWG** (не Legacy)
2. Нажми **Подключиться**
3. Проверь что IP изменился — открой в браузере https://2ip.ru

---

## Шаг 5. Поделись подключением с другими устройствами

Чтобы использовать VPN на других устройствах (телефон, планшет и т.д.):

1. В приложении нажми на подключение → **Поделиться**
2. Выбери **Поделиться через QR-код** или **Скопировать ключ**
3. На другом устройстве установи AmneziaVPN и отсканируй QR-код
   или вставь ключ через **Подключиться через ключ**

## Полезные команды на сервере

Проверить что контейнер AmneziaWG запущен:

```bash
docker ps | grep amnezia
```

Логи:

```bash
docker logs amnezia-awg
```

## Что делать если не подключается

1. **Проверь версию ядра** — должна быть 4.14 и выше:
   ```bash
   uname -r
   ```

2. **Проверь что UDP не заблокирован** у хостера. AmneziaWG работает
   только через UDP. Некоторые хостеры блокируют UDP или ограничивают
   его скорость. Проверить:
   ```bash
   nc -u -v 8.8.8.8 53
   ```
   Если зависает — UDP заблокирован, хостер не подходит.

3. **Проверь порт** — AmneziaWG по умолчанию использует случайный UDP-порт.
   Если файрвол сервера закрыт — открой все UDP-порты:
   ```bash
   ufw allow proto udp from any
   ```

4. **Переустанови протокол** — в приложении зайди в настройки подключения,
   удали AmneziaWG и установи заново.
